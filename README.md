# Norma Branch Details
This branch is specifically hard coded 0 and 30 degree single static blocks that emulate the static cube contact problem:

examples/single/static-solid/single_cube

examples/single/static-solid/single_cube_30_rot

This __should__ be the boundary conditions that the Schwarz contact problem should be applying to cube-1. 

Purely for debugging and not intended to be merged into main.
